# Echo Agent Guide

## Agent skills

### Issue tracker

Issues and PRDs are tracked in GitHub Issues via the `gh` CLI. External PRs are not a triage request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

This repo uses the default Matt Pocock engineering-skill triage labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, and `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This is a single-context repo. Read `CONTEXT.md` at the repo root and relevant ADRs under `docs/adr/` before architecture, debugging, TDD, or issue-planning work. See `docs/agents/domain.md`.
